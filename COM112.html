COM112 – Introduction to Digital Electronics

1. Number System

A number system is a method of representing numbers using symbols.

Examples: Decimal, Binary, Octal and Hexadecimal.

2. Decimal Number System

The decimal system uses ten digits (0–9).

Example: 125₁₀

3. Binary Number System

The binary system uses only two digits: 0 and 1.

Example: 1011₂

4. Octal Number System

The octal system uses digits 0–7.

Example: 157₈

5. Hexadecimal Number System

Uses digits 0–9 and letters A–F.

Example: 3AF₁₆

6. Number Conversion

Conversion can be performed between decimal, binary, octal and hexadecimal systems.

Example: 13₁₀ = 1101₂

7. Logic Gates

Logic gates perform logical operations on binary inputs.

  • AND
  • OR
  • NOT
  • NAND
  • NOR
  • XOR
8. Boolean Algebra

Boolean algebra is used to simplify logical expressions.

Examples:

  • A + 0 = A
  • A · 1 = A
  • A + A = A
9. Deriving Boolean Expressions

Boolean expressions can be derived from truth tables and logic circuits.

Example: Y = A'B + AB'

10. Logic Circuit Simplification

Expressions can be simplified using Boolean laws.

Example: AB + AB' = A

11. Arithmetic Circuits

Arithmetic circuits perform binary arithmetic operations.

12. Half Adder

A half adder adds two binary digits.

Sum = A XOR B

Carry = AB

13. Full Adder

A full adder adds three binary inputs: A, B and Carry-in.

Sum = A XOR B XOR Cin

Carry = AB + ACin + BCin
