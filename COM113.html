COM113 – Introduction to Programming

1. Introduction to Programming

Programming is the process of writing instructions that a computer can execute to solve problems.

2. General Concept of Programming

Programming involves analyzing a problem, designing a solution and implementing it using a programming language.

3. Program and Programmer

A program is a set of instructions for a computer, while a programmer is the person who writes the program.

4. Computer Program

A computer program contains instructions that direct the computer on what to do.

5. Categories of Programming Languages
LanguageDescription
Machine LanguageBinary instructions executed directly by the computer.
Assembly LanguageUses mnemonics such as ADD and MOV.
High-Level LanguageHuman-readable languages such as C, Java and Python.
6. Software Life Cycle

The software life cycle includes planning, analysis, design, coding, testing, implementation and maintenance.

7. Features of a Good Program
  • Accuracy
  • Efficiency
  • Readability
  • Maintainability
  • Reliability
8. System Development Life Cycle (SDLC)

The SDLC provides a structured approach to developing software systems.

9. Algorithm

An algorithm is a step-by-step procedure for solving a problem.

Characteristics: finiteness, definiteness, input, output and effectiveness.

10. Introduction to Flowchart

A flowchart is a graphical representation of an algorithm.

11. Flowchart Symbols
SymbolMeaning
OvalStart/Stop
RectangleProcess
DiamondDecision
ParallelogramInput/Output
12. Uses of Flowchart
  • Program planning
  • Communication
  • Documentation
  • Debugging
13. Advantages and Disadvantages of Flowchart

Advantages: Easy to understand, easy to modify, helps in debugging.

Disadvantages: Time-consuming and difficult for very large systems.
