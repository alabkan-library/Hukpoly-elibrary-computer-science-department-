MTH111 – Logic and Linear Algebra

1. Meaning of Logic

Logic is the study of correct reasoning.

It is widely used in mathematics, computer science and digital electronics.

2. Statements

A statement is a sentence that is either true or false.

Examples:

  • 2 + 2 = 4 (True)
  • 5 < 3 (False)

Types: Simple statement and Compound statement.

5. Tautology, Contradiction and Contingency
  • Tautology: Always true.
  • Contradiction: Always false.
  • Contingency: Sometimes true and sometimes false.
6. Basic Set Concepts

A set is a collection of distinct objects.

Example: A = {1,2,3,4}

Operations: Union, Intersection, Difference and Complement.

7. Introduction to Matrices
\[ 1 2 \]
\[ 3 4 \]

A matrix is a rectangular arrangement of numbers in rows and columns.

Example:

A = [1 2 ; 3 4\]

8. Matrix Addition and Subtraction

Matrices must have the same order.

A = [1 2 ; 3 4\]

B = [5 6 ; 7 8\]

A + B = [6 8 ; 10 12\]

9. Matrix Multiplication

If A is m × n and B is n × p, then AB is possible.

Example:

A = \[1 2 ; 3 4\], B = \[1 0 ; 0 1\]

AB = \[1 2 ; 3 4\]

10. Determinants of 2×2 Matrices

For A = \[a b ; c d\]

det(A) = ad − bc

Example:

A = \[1 2 ; 3 4\]

det(A) = (1×4) − (2×3) = −2

11. Vectors
y ↑ | | • (3,4) | / | / v | / ------+----------------→ x

A vector has both magnitude and direction.

Example: v = (3,4)

Magnitude = √(3² + 4²) = 5

12. Applications in Computer Science
  • Logic is used in programming and digital circuits.
  • Matrices are used in graphics and image processing.
  • Vectors are used in game development and physics simulations.
