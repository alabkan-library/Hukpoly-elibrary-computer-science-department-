COM115 – Computer Application Package

1. Introduction to Application Packages

Application packages are software programs designed to help users perform specific tasks such as typing documents, preparing calculations and making presentations.

Examples: Microsoft Word, Excel, PowerPoint and Access.

2. Word Processing

Word processing involves creating, editing, formatting and printing text documents.

Features: spell checking, page numbering, headers and footers.

3. Text Formatting
  • Bold
  • Italic
  • Underline
  • Font size
  • Font color
  • Alignment

5. Introduction to Spreadsheet

A spreadsheet is used for calculations, data analysis and chart creation.

The intersection of a row and column is called a cell.

7. Creating Charts in Excel

Charts are graphical representations of data.

  • Bar Chart
  • Column Chart
  • Pie Chart
  • Line Chart
8. Presentation Package

PowerPoint is used to create slide presentations.

Common features: themes, animations, transitions and slide shows.

9. Internet Applications

The internet allows users to access information, communicate and share resources.

Examples: Google Chrome, Firefox and Microsoft Edge.

10. Electronic Mail (E-mail)

E-mail is a system for sending and receiving electronic messages.

Steps to send an email:

  1. Open your email account
  2. Click Compose
  3. Enter recipient address
  4. Type subject
  5. Write the message
  6. Click Send
