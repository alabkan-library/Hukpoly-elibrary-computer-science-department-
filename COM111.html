COM111 – Introduction to Computer Science

1. What is a Computer?

A computer is an electronic device that accepts data, processes it, stores it and produces useful information.

Example: Desktop computer, laptop, tablet.

2. Basic Functions of a Computer

The five basic functions are:

  1. Input
  2. Processing
  3. Storage
  4. Output
  5. Control
3. Input Devices

Input devices are used to enter data into a computer.

Examples: Keyboard, mouse, scanner, microphone, webcam.

4. Output Devices

Output devices present processed information to the user.

Examples: Monitor, printer, speakers, projector.

5. Main Parts of a PC

The main parts include:

  • System Unit
  • Monitor
  • Keyboard
  • Mouse
  • Power Supply
